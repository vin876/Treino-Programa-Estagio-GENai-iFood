O que são agentes?
# Agentes são sistemas baseados em IA que conseguem:
# Receber um objetivo
# Planejar passos
# Executar ferramentas (código, funções, APIs)
# Iterar até resolver o problema


Como o iFood usa agentes:
# Otimização de rotas e entregas
# Atendimento com automação
# Análise interna de dados
# Plataforma Toqan para criar fluxos de agentes


Arquitetura de agente simples:
usuário -> objetivo -> planner -> executor -> ferramentas -> resposta final
